# 森林损失情景控制与蒙特卡洛碳储量模拟系统 V1.0

简称：森林损失控制碳模拟系统

英文名称：Forest Loss Scenario Control and Monte Carlo Carbon Stock Simulation System V1.0

本系统围绕森林损失情景控制、扰动事件生成、扰动强度赋值、碳储量预测、蒙特卡洛不确定性模拟和结果导出组织功能。

## 运行环境

- Python 3
- Streamlit
- NumPy
- Pandas
- PyYAML
- scikit-learn
- rasterio
- joblib

依赖安装：

```powershell
python -m pip install -r requirements.txt
```

## 启动方式

Windows 下可以双击：

```text
start_system.bat
```

也可以在 PowerShell 中运行：

```powershell
.\start_system.ps1
```

手动启动：

```powershell
python run_system.py
```

启动后访问：

```text
http://localhost:8501
```
