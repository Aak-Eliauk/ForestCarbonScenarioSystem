# 森林损失情景控制与蒙特卡洛碳储量模拟系统 V1.0 软件说明书

简称：森林损失碳模拟

英文名称：Forest Loss Scenario Control and Monte Carlo Carbon Stock Simulation System V1.0

## 1. 软件概述

森林损失情景控制与蒙特卡洛碳储量模拟系统 V1.0 面向森林损失控制和未来碳储量模拟场景，支持用户配置森林损失、采伐扰动、城镇扩张和保护区约束等情景参数，生成未来扰动事件，执行碳储量模拟，并输出统计结果、栅格结果和运行日志。

## 2. 功能模块

1. 数据准备模块：管理 AGBD、TCC、LULC、驱动因子、保护区和环境因子等输入数据。
2. 情景控制模块：设置森林损失控制、采伐面积控制、城镇扩张控制、扰动强度削减和目标年份。
3. 事件生成模块：生成采伐扰动、城镇转换和城镇边缘扰动事件。
4. 强度计算模块：为扰动事件分配强度，并支持不同强度方法。
5. 蒙特卡洛模拟模块：执行多次碳储量模拟，形成均值、分位数和不确定性结果。
6. 结果展示模块：展示指标表、分布统计、GeoTIFF 输出路径和运行日志。

## 3. 运行流程

1. 启动系统并进入浏览器界面。
2. 选择演示模式或真实栅格模式。
3. 配置输出目录、输入数据路径和情景参数。
4. 执行运行检查。
5. 启动快速测试或完整模拟。
6. 查看结果摘要、蒙特卡洛分布和输出文件。
7. 导出结果表格和栅格文件。

## 4. 输入输出

主要输入包括基准年 AGBD 栅格、树冠覆盖度栅格、土地利用/覆被栅格、目标年情景栅格、森林损失驱动因子、保护区掩膜、环境因子和情景参数。

主要输出包括情景汇总表、蒙特卡洛分布表、模型训练摘要、年度事件统计、平均 AGBD 栅格、平均 AGC 栅格和分位数栅格。

## 5. 运行环境

系统基于 Python 和 Streamlit 实现，依赖 NumPy、Pandas、PyYAML、scikit-learn、rasterio 和 joblib 等第三方库。
