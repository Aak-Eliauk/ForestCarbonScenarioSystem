# 森林损失情景控制与蒙特卡洛碳储量模拟系统 V1.0 软件说明书

## 1. 运行环境

本系统基于 Python 3 和 Streamlit 运行，主要依赖 NumPy、Pandas、PyYAML、scikit-learn、rasterio、joblib 等第三方库。

## 2. 安装依赖

进入系统目录后执行：

```powershell
python -m pip install -r requirements.txt
```

## 3. 启动系统

Windows 下可双击：

```text
start_system.bat
```

也可以在 PowerShell 中执行：

```powershell
.\start_system.ps1
```

或手动启动：

```powershell
python run_system.py
```

启动完成后，在浏览器访问终端中显示的本地地址，通常为：

```text
http://localhost:8501
```

如默认端口被占用，系统会自动改用后续可用端口，并在终端显示新的访问地址。

## 4. 基本使用

打开系统页面后，按界面步骤完成数据准备、情景方案设置和运行检查，然后选择快速测试或完整运行。运行完成后，可在结果页面查看统计表、分布结果、栅格输出路径和运行日志。
