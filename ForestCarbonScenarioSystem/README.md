# 森林损失情景控制与蒙特卡洛碳储量模拟系统 V1.0

简称：森林损失碳模拟

英文名称：Forest Loss Scenario Control and Monte Carlo Carbon Stock Simulation System V1.0

本项目为软件著作权申报主体工程，围绕森林损失情景控制、扰动事件生成、扰动强度赋值、碳储量预测、蒙特卡洛不确定性模拟和结果导出组织功能。

## 运行环境

- Python 3
- Streamlit
- NumPy
- Pandas
- PyYAML
- scikit-learn
- rasterio
- joblib

依赖安装：

```powershell
python -m pip install -r requirements.txt
```

## 启动方式

Windows 下可以双击：

```text
start_system.bat
```

也可以在 PowerShell 中运行：

```powershell
.\start_system.ps1
```

手动启动：

```powershell
python run_system.py
```

启动后访问：

```text
http://localhost:8501
```

## 申报范围

本次软著申报主体限定为当前目录：

```text
I:\ForestCarbonSoftWork\ForestCarbonScenarioSystem
```

`ForestCarbonScenarioSystem_SourcePack`、输出目录、缓存文件、测试运行产物和数据文件不属于本次申报主体。
